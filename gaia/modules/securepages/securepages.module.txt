Secure Pages (http://drupal.org/project/securepages)
----------------------------------------------------

Description
===========

Patch solves various redirection problems:
1. The same address being redirected back and forth to http and https (e.g. user/[uid]/edit).
2. Pages incorrectly served via httpd.
3. Pages incorrectly served via http.

These problems were occurring randomly (?) as a user surfed the site.

Compatibility
=============

6.x-1.9 (last release).

==============
